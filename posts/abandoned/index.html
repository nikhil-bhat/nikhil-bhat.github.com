Old Game on bhagat singh

February 18, 2020 1-minute read
games • howto • linux
linux • wine • game • bhagat • singh

Recently came across this article

Took me back to good old college days when my friend had demoed this game to me.

The novelty factor about this game being Hindi dialogue.

I wanted to relive the experience and luckily, i found the game on an abandonware site.

Here I have jotted down the steps to actually play this game on Linux which is my OS of choice

wget https://dl.myabandonware.com/t/37eh49RPy2DEroOS85U1CIF6w3b611vERgHURK0A3p5dl53Yc6/Bhagat-Singh-The-Game_Win_EN_ISO-Version.zip
sudo apt install -y bchunk wine64
unzip -d bhagat Bhagat-Singh-The-Game_Win_EN_ISO-Version.zip
sudo mkdir /mnt/bhagat &&  sudo mount bhagat.iso /mnt/bhagat/ -o loop,ro
cd /mnt/bhagat && wine Start.exe
